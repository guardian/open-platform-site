Overview

The Content API is a public service for accessing all the content the Guardian creates and the collections we have of that content (tags and sections). There are over one and a half million items available published as far back as 1999.

To access the API, you will need to sign up for an API key, which should be sent with every request. We will use your contact details to give you notice of any upcoming changes.

The easiest way to see what data is included is to use the Content API Explorer. You can use this tool to build complex queries quickly and browse the results. We also have a new beta Explorer which is more suitable for non-developers.

If you have any problems you can speak to other developers at the Guardian API talk group or tweet us at @openplatform.
